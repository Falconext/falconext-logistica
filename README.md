
# Logistica App

Monorepo for Logistica Application.

## Stack
- **Apps**:
  - `web`: Next.js 14 (Frontend)
  - `api`: NestJS (Backend)
- **Database**: SQLite (Prisma)
- **Tools**: Turbo, pnpm

## Getting Started

1. Install dependencies:
   \`\`\`bash
   pnpm install
   \`\`\`

2. Start development:
   \`\`\`bash
   pnpm dev
   \`\`\`
